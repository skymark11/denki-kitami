## 戦略策定テンプレート

新しいサービスの戦略書は以下の構成で作成する:

```markdown
# {サービス名} ページ コンテンツ戦略

## 1. ページ概要
- サービス名:
- URL: /service/{slug}/
- ターゲット文字数: 3,500〜5,000字
- 目標検索順位: 3位以内（6ヶ月後）

## 2. ターゲットキーワード
### メインKW（タイトル・H1に使用）
- {KW1} — 月間検索Vol: xxx

### サブKW（H2・本文に使用）
- {KW2}, {KW3}, ...

### ロングテール（FAQ・本文に自然配置）
- {KW4}, {KW5}, ...

## 3. 検索意図の分類
- Do（行動）: 「{地域} {サービス} 業者」→ 料金・CTA重視
- Compare（比較）: 「{サービス} 費用 相場」→ 料金表・他社比較
- Know（知識）: 「{サービス} とは」→ 解説・FAQ重視

## 4. ターゲットペルソナ
- ペルソナ1: {年齢層・居住形態・悩み}
- ペルソナ2: ...

## 5. 差別化要素
- 北見市/地域密着: {具体的な強み}
- 寒冷地対応: {該当する場合}
- 価格透明性: {料金表の充実}
- 即日対応: {緊急性がある場合}

## 6. 必須共起語（30語以上）
### サービス・設備関連（15語）
- ...
### 費用関連（10語）
- ...
### 差別化要素（5語）
- ...

## 7. コンテンツ構成
{Hero型/PageTitle型 に応じたセクション構成}

## 8. KPI目標
- 検索順位: {メインKW} 3位以内
- 月間流入: {目標}PV
- 問い合わせ: 月{目標}件
```

## コンプライアンス注意事項
- 景品表示法: 根拠のない最上級表現は禁止
- 建設業許可: 保有していないため表示禁止
- 実績数値: 虚偽の数値は一切禁止
- 価格表示: 必ず「〜」を付けて目安であることを明示
