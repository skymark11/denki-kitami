## 1. 景品表示法の監査

### 1.1 優良誤認表示（不当な品質・規格表示）

#### ⚠️ 要改善：価格表示の根拠不明確
**該当箇所:**
- 6行目: `title`タグ「【69,800円〜】」
- 52行目: 「200Vコンセント設置 69,800円〜」
- 107行目: 「総額目安 69,800円〜」
- 249行目: 料金表「総額目安 69,800円〜」

**問題点:**
「〜」表記を使用している価格表示について、最低価格を表示しているが、追加費用が発生する条件が複数存在する（288-293行目）。

**法的リスク:**
景品表示法第5条第1号に基づき、実際の取引条件よりも著しく有利と誤認させる可能性がある。

**改善提案:**
```html
<!-- 改善例 -->
<span class="price-amount"><span class="price-num">69,800</span>円〜</span>
<span class="price-note">（税込・基本工事費込み・配線15m以内）</span>
```

価格表示の直近に以下の条件を明示：
- 配線距離15m以内
- 分電盤に空き回路がある場合
- 契約電力の変更が不要な場合

## 2. 特定商取引法の監査

### 2.1 事業者情報の明記

#### ❌ 重大な問題：事業者情報の不足
**該当箇所:**
- 596-600行目: フッター（会社情報）

```html

<p>〒090-XXXX 北海道北見市○○町X丁目X-X</p>

**問題点:**

4. **事業者の法人番号の記載なし**

    <p>Email: info@denken-kitami.co.jp</p>
    <p>法人番号: 1234567890123</p>
    <p>北海道知事登録 第[登録番号]号</p>
    <p>営業時間：8:00〜18:00（年中無休）</p>
</div>
```

## 8. まとめ

### 現状の問題点
本Webページは、**特定商取引法**、**個人情報保護法**、**電気工事業法**の3つの法令において重大な不適合が確認されました。特に以下の点が緊急の対応を要します：

1. 事業者情報が仮の表示（住所、電話番号が「XX」表記）

4. キャンセル・返品条件が一切記載されていない

### 法的リスク
- 特定商取引法違反により、業務改善命令または100万円以下の罰金のリスクがあります。
- 個人情報保護法違反により、個人情報保護委員会からの勧告・命令のリスクがあります。
- 電気工事業法違反により、行政指導のリスクがあります。

### 推奨対応
**即座に以下の対応を実施してください：**
1. 仮の情報（XX表記）を実際の情報に差し替え
2. プライバシーポリシーページを作成し、フォームに同意取得を追加
3. 電気工事業登録番号と資格者情報を追加
4. 特定商取引法に基づく表記ページを作成

**上記対応後、以下の改善を順次実施してください：**
5. キャンセル・返品規定の作成
6. 価格表示の条件明確化
7. HTTPS化（SSL証明書導入）
