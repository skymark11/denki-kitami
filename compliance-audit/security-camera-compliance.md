## 1. 景品表示法（不当表示防止）

### 1.1 最上級表現の使用チェック

#### ✅ 確認事項：「最安値」「No.1」等の根拠なき最上級表現

**チェックポイント:**
- 「最安値」「地域最安」「業界最安」等の表現がないか
- 「No.1」「第1位」「実績トップ」「設置台数トップ」等の表現がないか
- 「最高画質」「最速設置」「最高性能」等の根拠なき最上級表現がないか
- 「北見で一番選ばれている」等の根拠なき優位性表現がないか

**注意事項:**
これらの表現を使用する場合は、景品表示法第5条第1号（優良誤認表示）に基づき、客観的な調査データ等の合理的根拠が必要です。根拠資料は消費者庁から求められた場合に15日以内に提出できるよう保管してください（不実証広告規制）。

**改善提案:**
最上級表現を使用する場合は以下のように根拠を明示：
```html
<!-- 不適切な例 -->

<!-- 適切な例 -->
<p>電気工事士の資格を持つ専門スタッフが、ご自宅に最適な防犯カメラをご提案いたします</p>
```

### 1.2 価格表示の適正性

#### ⚠️ 要改善：基本料金表示の条件明確化

**チェックポイント:**
- 税込表示がされているか → 「税込」と明記されていれば適正
- 「〜」表記の条件が明確か
- カメラ本体代と工事費の区分が明確か

**問題点:**
防犯カメラの設置費用は、カメラ本体代・録画装置代・配線工事費・設定費など複数の要素で構成されます。基本料金の表示において、何が含まれ何が別途必要かが不明確な場合、有利誤認表示（景品表示法第5条第2号）に該当する可能性があります。

**法的リスク:**
消費者が表示価格のみで全ての工事・機器が揃うと誤認する恐れがある。特にカメラ本体・録画装置（NVR/DVR）・HDD等の機器代が別途必要な場合、その旨の明記が必須。

**改善提案（必須）:**
```html
<!-- 改善例 -->
<span class="price-amount"><span class="price-num">22,000</span>円〜</span>
<span class="price-note">（税込・カメラ1台設置工事費のみ・機器代別途）</span>

<!-- 価格の直近に条件を明示 -->
<ul class="price-conditions">
    <li>※上記は既存コンセントから電源を取る場合の設置工事費です</li>
    <li>※カメラ本体・録画装置（NVR/DVR）・HDD等の機器代は別途かかります</li>
    <li>※新規電源配線工事が必要な場合は追加費用が発生します</li>
    <li>※高所作業（2階以上の外壁等）は別途高所作業費がかかる場合があります</li>
    <li>※正確な費用は現地調査後にお見積もりいたします</li>
</ul>
```

## 3. 電気工事業法・電気工事士法

### 3.1 防犯カメラ設置における資格要件の明示

#### ❌ 重大な問題：資格要件の説明不足

**法的背景:**
防犯カメラの設置工事には、作業内容によって電気工事士資格の要否が異なります。この区別をページ上で正確に記載する必要があります。

**資格要件の整理:**

| 作業内容 | 電気工事士資格 | 根拠法令 |
|---------|-------------|---------|
| AC100V電源への直結配線 | **必要**（第二種以上） | 電気工事士法第3条 |
| コンセントの新設・増設 | **必要**（第二種以上） | 電気工事士法第3条 |
| 壁内・天井裏の隠蔽配線 | **必要**（第二種以上） | 電気工事士法第3条 |
| 分電盤からの専用回路増設 | **必要**（第二種以上） | 電気工事士法第3条 |
| カメラ本体の取付のみ（ビス止め等） | 不要 | 電気工事士法施行令第1条 |
| PoE（LAN給電）カメラの接続 | 不要 | 電気工事士法施行令第1条 |
| 既存コンセントへのACアダプター接続 | 不要 | 電気工事士法施行令第1条 |
| Wi-Fiカメラの設置・設定 | 不要 | 電気工事士法施行令第1条 |

**改善提案（必須）:**
以下の注意書きをページに明記してください：
```html
<div class="qualification-notice">
    <h4>防犯カメラ設置工事の資格要件について</h4>
    <p><strong>※AC100V電源への直結配線、コンセント新設、壁内の隠蔽配線を伴う
    防犯カメラ設置工事は、電気工事士法に基づき、電気工事士の資格が必要です。</strong>
    無資格での施工は法律違反となり、感電や火災の原因となる恐れがあります。</p>
    <p>※カメラ本体のビス止めやPoE給電のみの設置は電気工事士資格を要しませんが、
    防水処理・配線の安全確認・最適な設置角度の調整のため、専門業者への依頼を推奨いたします。</p>
    <p>当店では第二種電気工事士の資格を持つスタッフが施工いたします。</p>
</div>
```

## 4. 建設業法

### 4.1 建設業許可の要否

#### ✅ 問題なし：軽微な工事に該当

**根拠:**
建設業法第3条第1項ただし書き及び同法施行令第1条の2に基づき、工事1件の請負代金が500万円未満（税込）の場合は建設業許可が不要です。

**確認事項:**
防犯カメラ設置工事は、一般住宅向けの場合、機器代込みでも数万円〜数十万円程度であり、500万円を超えることは一般的に想定されません。したがって、建設業許可がなくても施工可能です。

**注意事項:**
- 当事業者は建設業許可を保有していないため、「建設業許可取得済み」等の表記は行わないこと
- 万一、商業施設・工場等の大規模カメラシステム（多数台の設置＋ネットワーク構築等）で500万円以上になる案件は受注できません
- 500万円以上の案件の問い合わせがあった場合は、建設業許可を持つ事業者を紹介する対応とすること

**ページ上での記載例:**
```html
<p class="note">※当店は500万円未満の軽微な電気工事を専門としております。
大規模なカメラシステム構築（500万円以上）は対応範囲外となります。</p>
```

## 5. 個人情報保護法

> **★ 防犯カメラページにおいて最も重要な法令対応セクション**
> 防犯カメラは他のサービスと異なり、第三者（通行人・近隣住民・来訪者等）の個人情報を継続的に取得・記録する設備です。設置を推奨・施工する事業者として、顧客に対して個人情報保護法およびプライバシー権に関する適切な案内を行う責任があります。

### 5.1 防犯カメラ映像と個人情報保護法

#### ❌ 重大な問題：防犯カメラ映像の個人情報としての取扱い案内

**法的背景:**
防犯カメラで撮影された映像のうち、特定の個人を識別できるもの（顔が判別できる映像等）は、個人情報保護法第2条における「個人情報」に該当します。防犯カメラを設置するお客様は個人情報保護法上の「個人情報取扱事業者」に該当する可能性があり、以下の義務が発生します。

**お客様（カメラ設置者）に案内すべき事項:**

| 義務 | 内容 | 根拠条文 |
|------|------|---------|
| 利用目的の特定 | 「防犯目的」等、撮影の目的を明確にする | 第17条 |
| 利用目的の通知・公表 | 撮影中であることの掲示（ステッカー等） | 第21条 |
| 安全管理措置 | 映像データの適切な保管・アクセス制限 | 第23条 |
| 第三者提供の制限 | 映像の目的外利用・第三者への提供制限 | 第27条 |
| 保有個人データの開示 | 本人からの開示請求への対応 | 第33条 |

**改善提案（必須）:**
```html
<div class="privacy-notice-camera">
    <h4>防犯カメラ設置に伴う個人情報の取扱いについて</h4>
    <p>防犯カメラの映像は個人情報保護法上の「個人情報」に該当する場合があります。
    カメラ設置者（お客様）には以下の対応をお願いしております：</p>
    <ul>
        <li><strong>撮影中の掲示:</strong> カメラの設置場所付近に
        「防犯カメラ作動中」等のステッカー・看板を掲示してください
        （個人情報保護法第21条）</li>
        <li><strong>撮影範囲の限定:</strong> 撮影範囲は必要最小限に留め、
        隣家の敷地内や公道を不必要に撮影しないようにしてください</li>
        <li><strong>映像データの管理:</strong> 録画データへのアクセスを制限し、
        不要になったデータは速やかに削除してください</li>
        <li><strong>目的外利用の禁止:</strong> 撮影映像は防犯目的以外に使用しないでください</li>
        <li><strong>第三者への提供:</strong> 警察等の捜査機関から法令に基づく要請があった場合を除き、
        映像を第三者に提供しないでください</li>
    </ul>
    <p>※当店では「防犯カメラ作動中」ステッカーの設置を施工時にご案内しております。</p>
</div>
```

### 5.2 プライバシー権・肖像権への配慮

#### ❌ 重大な問題：隣家・公道への撮影範囲に関する注意喚起

**法的背景:**
防犯カメラの設置に際し、撮影範囲が隣家の敷地や公道に及ぶ場合、以下の法的リスクが発生します：

- **プライバシー権の侵害（民法第709条・第710条）:** 隣家の私的領域を継続的に撮影する行為は、プライバシー権の侵害として不法行為に基づく損害賠償請求の対象となりえます
- **肖像権の侵害:** 公道を通行する人物を継続的に撮影・記録する行為は、肖像権侵害として問題になる場合があります
- **近隣トラブルの原因:** 防犯カメラの設置は近隣住民との信頼関係を損なう可能性があり、設置前の配慮が重要です

**判例等の参考:**
- 防犯カメラの撮影範囲が隣家に向けられ、プライバシー侵害として撤去命令が出された裁判例が存在します
- 撮影の必要性・相当性（犯罪被害の事実等）と、プライバシー侵害の程度を比較衡量して判断されます

**改善提案（必須）:**
```html
<div class="camera-privacy-notice">
    <h4>撮影範囲とプライバシーへの配慮</h4>
    <ul>
        <li><strong>撮影範囲の調整:</strong> カメラの設置角度は、ご自宅の敷地内を中心に
        撮影するよう調整いたします。隣家の敷地内が大きく映り込む設置は避けてください</li>
        <li><strong>公道の撮影:</strong> 防犯上やむを得ず公道が映り込む場合は、
        撮影範囲を必要最小限に調整いたします</li>
        <li><strong>近隣への配慮:</strong> 可能であれば、防犯カメラの設置について
        近隣の方にお知らせいただくことをお勧めいたします</li>
        <li><strong>マスキング機能:</strong> 隣家の窓や玄関等が映り込む場合は、
        プライバシーマスキング機能（対応機種の場合）で該当箇所を非表示にすることも可能です</li>
    </ul>
    <p>※設置場所のご相談の際に、撮影範囲とプライバシーへの配慮について
    丁寧にご説明いたします。</p>
</div>
```

#### ⚠️ 要改善：自治体ガイドラインへの言及

**法的背景:**
多くの自治体では、防犯カメラの設置・運用に関するガイドラインを策定しています。法的拘束力は条例化されていない限り強制力はありませんが、適切な運用の指針として参照すべきです。

**主な自治体ガイドラインの共通事項:**
- 設置目的の明確化（防犯目的に限定）
- 撮影範囲の必要最小限化
- 「防犯カメラ作動中」等の掲示義務
- 録画データの保存期間の設定（通常7〜30日程度）
- 録画データの管理責任者の指定
- 映像データの第三者提供の制限
- 苦情対応窓口の設置

**改善提案:**
```html
<div class="guideline-notice">
    <h4>防犯カメラ設置に関するガイドライン</h4>
    <p>防犯カメラの設置・運用にあたっては、各自治体のガイドラインに沿った
    運用をお勧めいたします。北海道では「北海道防犯カメラの設置及び運用に関する
    ガイドライン」等が策定されています。</p>
    <p>設置時に、適切な運用方法についてご案内いたします。</p>
</div>
```

## 6. コンテンツ表現チェック

### 6.1 不当な比較広告

#### ✅ 問題なし（要確認）

**チェックポイント:**
以下のような不当な比較広告がないか確認してください：
- 「他社より○○円安い」（根拠不明の比較）
- 「大手警備会社より安い」（客観的根拠のない比較）
- 「ホームセンターより高品質」（客観的根拠のない比較）
- 特定の競合他社・警備会社を名指しで批判する表現

**注意事項:**
比較広告を行う場合は、公正取引委員会「比較広告に関する景品表示法上の考え方」に基づき、客観的データに基づく実証可能な比較であることが必要です。

### 6.2 「安心」「安全」の多用チェック

#### ⚠️ 要改善：根拠のない抽象的表現

**チェックポイント:**
「安心」「安全」「信頼」等の抽象的表現が根拠なく多用されていないか確認してください。防犯カメラのページでは特に「安心の防犯対策」「安全な暮らし」等の表現が多用されがちです。

**問題となる例:**
```html
<!-- 不適切 -->
<p>安心・安全の防犯カメラ設置</p>
<p>安心の24時間監視</p>
<p>安全な暮らしをお約束します</p>
```

**改善提案:**
抽象的な表現の代わりに具体的な根拠を示してください：
```html
<!-- 適切 -->
<p>電気工事士の有資格者が防水処理・配線を含めて確実に施工</p>
<p>施工後1年間の工事保証付き</p>
<p>現地調査・お見積もり無料、追加費用なしの明朗会計</p>
<p>200万画素以上の高解像度カメラで夜間も鮮明に撮影</p>
```

### 6.3 煽り表現・緊急性の演出

#### ⚠️ 要改善：不当な緊急性の演出がないか

**チェックポイント:**
以下のような過度な煽り表現がないか確認してください：
- 「空き巣被害が急増中！今すぐ防犯カメラを！」（根拠不明の統計）
- 「カメラがなければ犯罪被害のリスクが○倍！」（根拠不明の数値）
- 「期間限定キャンペーン」（常時実施している場合は不当表示）
- 「この地域で被害が多発しています」（根拠不明の地域情報）

**改善提案:**
事実に基づいた表現を使用：
```html
<!-- 適切な例 -->
<p>防犯カメラの設置は、不審者への抑止効果が期待でき、
万一の際の映像記録としても役立ちます。</p>
<p>北海道警察の統計によると、防犯カメラの設置は犯罪抑止に
一定の効果があるとされています。</p>
<!-- ※統計データを引用する場合は必ず出典を明記すること -->
```

### 6.5 写真・画像の適正使用

#### ✅ 問題なし（要確認）

**チェックポイント:**
- メーカーの製品画像を使用する場合は、メーカーの許諾を得ているか
- 施工事例の写真は顧客の許諾を得ているか

- **カメラの映像サンプルを掲載する場合、個人が特定できる映像でないか**
- ストックフォトを使用する場合はライセンス範囲内か

## 7. 総合評価・改善提案

### 7.1 総合評価

本防犯カメラ設置工事ページに対する法令監査の結果、以下の対応が必要です。特に個人情報保護法・プライバシー権に関する対応は、防犯カメラサービス固有の重要事項であり、最優先で取り組む必要があります。

### 7.2 改善優先度と実施ロードマップ

#### 【緊急】即座に対応が必要

1. **防犯カメラ映像の個人情報としての取扱い案内を記載**
   - 撮影中の掲示義務（ステッカー設置案内）
   - 映像データの管理・削除に関する注意事項
   - 目的外利用・第三者提供の制限
   - **期限:** 即日

2. **プライバシー権・肖像権への配慮記載**
   - 撮影範囲の調整に関する案内（隣家・公道への配慮）
   - プライバシーマスキング機能の案内
   - 近隣住民への配慮事項
   - **期限:** 即日

3. **電気工事士資格の要件を明記**
   - 資格が必要な作業と不要な作業の区別を明示
   - 無資格施工の危険性を注意喚起
   - **期限:** 即日

4. **特定商取引法対応（事業者情報・キャンセル条件）**
   - 事業者の正確な情報を記載
   - キャンセル・返品・保証条件を明記
   - **期限:** 即日

5. **価格表示の条件明確化**
   - 工事費と機器代の区分を明示
   - 追加費用の発生条件を記載
   - **期限:** 即日

#### 【重要】早急に対応が必要

6. **自治体ガイドラインへの言及**
   - 北海道の防犯カメラ設置運用ガイドラインの参照案内
   - 録画データ保存期間の目安の記載
   - **期限:** 1週間以内

7. **プライバシーポリシーの整備**

    - **期限:** 2週間以内

    - 顧客の掲載許可の取得・保管
    - 施工事例写真における第三者のプライバシー保護の確認
    - **期限:** 1ヶ月以内
